
## 기능 명세
### USER
- 로그인 기능
    - basic token기반 세션 로그인
      -회원가입
- 내가 대출중인 도서 조회 ( 연체금 조회 )
- 전체 회원 조회
- 비밀번호 재설정
- 회원 탈퇴

### 도서
- 도서 등록
- 도서 삭제
- 도서 정보 수정
- 제목, 
  - 작가, 출판사 (작가 및 출판사의 경우 미래 테이블 분리를 고민해 봐야 함.)
- 대출 랭킹
- 도서 목록 조회

### 대출
- 사용자 대출 도서 등록
- 전체 대출 현황 조회
- 정상 대출에서 연체 대출 상태로 변경
    - 대출 기본 일주일, 대출 일주일 연장 1번 가능
- 반납 완료
- 사용자별 도서 연체금 및 전체 연체금 총합 조회
